# Echohive - Mini Social Media Platform

## 📌 Description
Echohive is a mini social media web application that allows users to share their thoughts, images, and moods in real-time. The platform is designed to simulate core social networking features such as posting,chating, interacting, and user profiles, providing a dynamic and engaging user experience.

## 🚀 Features
- User authentication (Signup / Login)
- Create, edit, and delete posts
- Share text and images
- Like and comment on posts
- chats
- Personalized news feed
- User profile with posts history
- Follow / unfollow users
- Search functionality

## 🛠️ Technologies Used
### Frontend:
- React.js
- HTML5
- CSS3
- JavaScript (ES6)

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB

### Other Tools:
- REST APIs
- JSON Web Token (JWT) for authentication
